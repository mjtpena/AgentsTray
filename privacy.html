Privacy Policy

Last updated: March 3, 2026

Overview

Agent Tray ("the App") is a Windows desktop application that monitors AI agent usage. Your privacy is important to us. This policy explains what data the App accesses and how it is handled.

Data Collection

Agent Tray does not collect, transmit, or store any personal data on external servers. All data remains on your local machine.

Data Stored Locally

The App stores the following data locally on your device:

  • Application settings — your provider preferences, alert thresholds, and UI preferences are saved in a JSON file at %LocalAppData%\AgentTray\settings.json.
  • API credentials — API keys and authentication tokens you provide are stored securely in Windows Credential Manager, the operating system's built-in secrets vault. They are never written to plain-text files.
  • Usage data — usage snapshots returned by provider APIs are held in memory only for display purposes and are not persisted to disk.

Network Requests

The App makes outbound HTTPS requests only to the provider APIs you have enabled (e.g., Anthropic, GitHub, OpenAI) to retrieve your usage data. No data is sent to any server operated by the App developer.

Cookies & Browser Data

For providers that use browser-based authentication (e.g., Cursor), the App may read authentication cookies from your local browser profile. This data is read locally and is never transmitted externally.

Third-Party Services

The App does not integrate with any analytics, advertising, or tracking services. There are no telemetry or crash-reporting frameworks.

Children's Privacy

The App is not directed at children under the age of 13 and does not knowingly collect information from children.

Changes to This Policy

We may update this privacy policy from time to time. Changes will be posted on this page with an updated "Last updated" date.
